thisWay

...find out how to get to any landmark!


thisWay is an app that tells you how to get from A to B. It allows you to input your email, an image link showing a landmark, and your start point (in the format of "city, country"). It will then use some fancy computer stuff to give you directions if you want to go see it. It might cost a lot to actually go, but hey, still works.
